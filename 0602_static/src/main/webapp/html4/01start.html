1. 주석 :
- 주의 사항 : html 주석은 웹 브라우저의 소스 보기에서 보인다.
- html 주석으로는 보안에 치명적일 수 있는 내용은 쓰면 안 된다.

2. 글자간 공백 :        공백 : & nbsp;
- 글자간 공백은 하나만 허용하므로, & n b s p ;를 사용하여 글자간 공백 확보.

3. 문단 구분 : p 태그, 줄바꿈은 br 태그.

4. html의 구조 : 시작 태그와 종료 태그의 한 쌍으로 이루어짐.
- html과 /html 사이에 코딩한다.
- 일부 태그는 종료 태그가 없다. (br, meta, hr)

5. html 내부는 head와 body로 구성.
- head : 문서의 정보
- body : 문서의 내용

6. head의 요소
- meta : 문서 정보를 웹브라우저에 알리는 역할
- meta charset="UTF-8" : 웹 브라우저가 이 문서를 조립할 때 문자는 utf8 방식으로 조립해라.
- title : 문서 제목.
