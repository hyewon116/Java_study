jquery 기본 사용법 : 함수(function) 기반

$ : jquery의 시작

$() : html 객체 지정

$(document) : 현재 실행되는 html 문서

$(document).ready(); : 현재 실행되는 html 문서의 load가 완료되면..

function(){} : anonymous function : 이름 없는 함수 : 이름만 빼고 일반 함수와 동일한 기능을 갖는다.

$(document).ready(function(){}); : 현재 실행되는 html 문서의 load가 완료되면 이름 없는 함수를 실행하라.
