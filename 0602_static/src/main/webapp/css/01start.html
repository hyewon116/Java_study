css : cascading style sheet(style) : html 요소의 디자인

html : 문서 구조(head/body)와 내용(input/button/text) ex)계산기의 형태

javascript : 동작 ex)계산기의 계산 기능

css : 비주얼을 그럴듯하게 꾸미기

html, js, css는 서로 다른 파일에 저장 & 링크로 연결

->html(퍼블리셔), js(프론트 개발자), css(디자이너)의 작업을 분리하는 개념

inline css : html 코드에 css를 첨부 : 현재는 잘 안 쓰는 방법
